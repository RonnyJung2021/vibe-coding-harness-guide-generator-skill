# Vibe Coding 操作指南生成器

## 核心功能

接收任意自然语言任务描述，输出结构化的操作指南，包含：
- 清晰的阶段划分
- 可直接粘贴到 Agent 的指令
- 明确的验收标准
- Harness 工程原则对齐

## 触发方式

当用户输入类似以下内容时触发：
- "帮我创建一个 Todo 应用"
- "实现用户登录功能"
- "修复首页加载慢的问题"
- "从零开始搭建一个博客系统"

## 输出结构

```markdown
## 阶段 X：[阶段名称]

### X1. [步骤名称]

**目的**：[说明此步骤要解决的问题]

**Agent 输入框粘贴**：
```text
[完整的 Agent 指令]
```

**验收标准**：
- [可验证的检查项]

**Harness 对齐**：[对应原则说明]
```

## 设计原则

| 原则 | 说明 |
|------|------|
| 输入容错 | 支持多模态、不精确的自然语言输入 |
| Harness 对齐 | 任务契约、状态落盘、上下文裁剪、工具边界、轻量验收 |
| Agent 友好 | 提供可直接复制粘贴的指令 |
| 验证导向 | 每步明确验收标准 |
